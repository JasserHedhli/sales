# Sales Forecasting with LSTM and MLflow

## Overview
This project involves forecasting sales using a Long Short-Term Memory (LSTM) model. The entire workflow includes connecting to a data warehouse, cleaning and analyzing the data, building the LSTM model, and using MLflow for tracking experiments.
